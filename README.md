# test-ozon

Веб-приложение с индикатором прогресса и панелью управления:
- Ввод значения прогресса от 0 до 100
- Переключатель анимации вращения
- Переключатель видимости индикатора

## Модуль progress

Модуль progress реализует функцию createProgress(), которая создает и управляет круговым индикатором прогресса на основе SVG

### Как работает:

1. Инициализация: 
Принимает DOM-узел для монтирования и начальное состояние {value, animated, hidden}

2. Рендеринг: 
    Создает SVG-структуру с двумя кругами:
   - track - фоновая дорожка
   - progress - индикатор прогресса с анимацией

3. Cостояние:
   - value - значение прогресса
   - animated - включает/выключает CSS-анимацию вращения
   - hidden - скрывает/показывает индикатор

4. API: Возвращает объект с методами:
   - setValue(value) - установка значения
   - setAnimated(isAnimated) - управление анимацией
   - setHidden(isHidden) - управление видимостью
